# Smart-Attendance-System-Attendx-
This project is an AI powered smart attendance and classroom engagement analytics system. It is designed to automate and enhance the already existing attendance system (ex. Qr, fingerprint,etc) in educational institutions. The system of software and hardware uses face recognition to automatically mark attendance based on realtime face recognition of students present in the classroom. Unlike the old conventional systems, attendance is not marked at a single instance but tracked continuously  throughout the class time, enabling us to mark duration-based and partial attendance. The system also measures student attention levels using face orientation during the class to generate engagement information. Attendance id first marked as pending (during the class_ and later verifies by using post class AI rechecking to ensure fairness and accuracy  for all students. The platform includes dashboards for teachers and students. Dispute resolution support, proxy detection and risk prediction analysis. This project allows institutions to mark attendance transparently, securely, and data driven.
